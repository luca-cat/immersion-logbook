# immersion-logbook
A Japanese immersion tracker with RPG-inspired progression, daily 占い multipliers, and a balance system that adapts to your weaknesses. Built with Python, SQLite, and Rich.

# Dependency installation

This project uses `uv` to manage dependencies.
Install uv: see docs at https://docs.astral.sh/uv/

Install dependencies:
```
uv sync
```

Add deps:

```
uv add <dependency>
```

# Environment varialbes:
DB_PATH - sqlite3 path
